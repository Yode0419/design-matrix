## 如何使用這份檔案 How to Use

### 矩陣說明

矩陣由兩條軸組成，形成四個象限：

```
                    ▲ 想多做 WANT TO DO MORE
                    │
          [成長區]  │  [甜蜜點]
   仍在學習         │            我是專家
   STILL    ────────┼──────────  EXPERT
   LEARNING         │            AT THIS
          [考慮放棄]│  [委派區]
                    │
                    ▼ 想少做 WANT TO DO LESS
```

| 象限 | 英文 | 意義 |
|------|------|------|
| 🌱 成長區（左上） | Learning + Want More | 你還在學，但很想精進的技能 |
| ⭐ 甜蜜點（右上） | Expert + Want More | 你很擅長且熱愛的技能 → 核心競爭力 |
| ❓ 考慮放棄（左下） | Learning + Want Less | 還不熟又沒興趣，可以考慮不投入 |
| 🤝 委派區（右下） | Expert + Want Less | 你會，但寧願交給別人做 |

### 專家程度定義 Expert Level Scale

| 程度 | 說明 |
|------|------|
| 初學 Beginner | 概念了解，還沒實際做過 |
| 普通 Pretty Good | 成功做過幾次，可能需要協助 |
| 熟練 Really Good | 獨立完成過 7-8 次，很少需要協助 |
| 精通 Amazing | 專業操作超過 12 次，能解釋如何取得成果 |
| 專家 Expert | 成功超過 20 次，**能夠教導他人** |

## 自我評估步驟 Self-Assessment Steps

### Step 1 — 逐一審視每張卡片

對每張技能卡，問自己兩個問題：

**問題 A：我的程度？（橫軸）**
- 左邊（Still Learning）→ 初學 / 普通
- 右邊（Expert）→ 熟練 / 精通 / 專家

**問題 B：我想做多少？（縱軸）**
- 上方（Want More）→ 我希望在職涯中更多做這件事
- 下方（Want Less）→ 我不想花太多時間在這上面

> 💡 **注意**：「想多做」是理想狀態，不要被現在的工作影響！想想你在一個理想的職位上，會希望多做什麼？

### Step 2 — 在每張卡片後填入你的象限

在每張技能後面的 `[ ]` 中填入你的位置：
- `[⭐]` = 甜蜜點（Expert + Want More）
- `[🌱]` = 成長區（Still Learning + Want More）
- `[🤝]` = 委派區（Expert + Want Less）
- `[❓]` = 考慮放棄（Still Learning + Want Less）

## 下一步 Next Steps

完成矩陣後，思考以下問題：

1. **我的甜蜜點在哪個領域？** 是 UX Research 型、UI Design 型、還是全端 UX 型？
2. **我的成長優先順序是什麼？** 從成長區挑選 1-3 個重點技能，設定 3-6 個月的學習計畫
3. **我目前的工作有多少在「委派區」？** 如果太多，可能代表職位不夠符合你的優勢
4. **有哪些甜蜜點技能我沒有機會展現？** 考慮透過 side project 或內部提案來創造機會
