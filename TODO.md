# Design Skills Matrix — 待辦事項

## 🗄️ 資料儲存與匯出

- [ ] **JSON 匯出**：新增「匯出 JSON」按鈕，將每個技能的放置座標與象限區域輸出為 `.json` 檔案
- [ ] **JSON 匯入**：支援讀取已匯出的 `.json` 檔案，還原上次的放置結果
- [ ] **自動儲存**：目前已使用 `localStorage`，確認格式是否與未來 JSON schema 一致，必要時統一資料結構
- [ ] 定義 JSON schema（技能 id、x/y 百分比、象限、timestamp）

## 🎨 UI / UX 微調

### 操作體驗
- [ ] 選取卡片後，矩陣區域加入明顯的「可放置」提示（例如虛線邊框或背景色變化）
- [ ] 已放置的技能在側邊欄顯示勾選或灰化，讓使用者清楚知道哪些還沒放
- [ ] 支援點擊矩陣上的點來移動或刪除單一技能（目前可能需要全部重製才能調整）
- [ ] 拖曳移動（drag to reposition）已放置的技能點

### 結果判讀
- [ ] 結果模式下各象限加入數量統計（例如「甜蜜點：12 項」）
- [ ] 每個象限的技能列表可展開，列出該區域的所有技能名稱
- [ ] 技能點 hover 時顯示 tooltip（技能中英文名稱 + 所屬象限）
- [ ] 提供列印 / 截圖友善的版面（隱藏操作按鈕，保留矩陣與圖例）

### 視覺細節
- [ ] 各分類色彩在矩陣點上更明顯（目前小點顏色是否夠清晰？）
- [ ] 行動裝置版面優化（側邊欄與矩陣的排版在手機上是否可用）

## 🔧 功能擴充（後續考慮）

- [ ] 多份矩陣管理（例如「現在的我」vs「半年後目標」）
- [ ] 分享功能：將結果編碼至網址，方便分享給他人查看
