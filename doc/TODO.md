# Design Skills Matrix — 待辦事項

## 🗄️ 資料儲存與匯出

- [x] **JSON 匯出**：新增「匯出 JSON」按鈕，將每個技能的放置座標與象限區域輸出為 `.json` 檔案
- [x] **JSON 匯入**：支援讀取已匯出的 `.json` 檔案，還原上次的放置結果
- [x] **自動儲存**：使用 `localStorage` 持久化放置狀態與側邊欄摺疊狀態
- [ ] 定義 JSON schema（技能 id、x/y 百分比、象限、timestamp）

## 🎨 UI / UX 微調

### 操作體驗
- [x] 選取技能項目後，矩陣顯示虛線導引線（連接點到中心軸），讓 x/y 位置更易判讀
- [x] 已放置的技能在側邊欄顯示勾選（✓ 標記），讓使用者清楚知道哪些還沒放
- [x] 支援點擊矩陣上的點顯示刪除按鈕（✕），可刪除單一技能
- [x] 拖曳移動（drag to reposition）已放置的技能點
- [ ] 選取技能後，矩陣區域加入更明顯的「可放置」背景提示（目前已有導引線，但邊框尚無）

### 結果判讀
- [x] 結果模式下各象限 toggle 按鈕（Q1–Q4 / 危險地帶 / 甜蜜點），點擊高亮對應區塊並淡出其他圓點，帶即時計數
- [x] 高亮區塊旁顯示浮動白色說明卡片（label、引言、中文說明），位置依區塊動態調整
- [x] 甜蜜點與危險地帶以橢圓形視覺呈現，計數以橢圓方程式精確判定
- [ ] 每個象限的技能列表可展開，列出該區域的所有技能名稱
- [ ] 技能點 hover 時顯示 tooltip（技能中英文名稱 + 所屬象限）
- [ ] 提供列印 / 截圖友善的版面（隱藏操作按鈕，保留矩陣與圖例）

### 視覺細節
- [x] 各分類色彩重新設計（9 色各具鮮明色相），並支援 per-category 色彩選擇器（點色塊即可自訂，持久化至 localStorage）
- [x] 側邊欄分類計數改為「已放 / 總數」格式（例如 2/7）
- [x] 說明按鈕：點擊後於矩陣角落顯示縱軸與橫軸說明卡片（含四段專業程度描述），並在矩陣上以虛線色框標示四段專業程度區塊
- [x] 格線為 toggle 按鈕樣式，文案精簡為「格線」
- [ ] 行動裝置版面優化（側邊欄與矩陣的排版在手機上是否可用）

## 🔧 功能擴充

- [x] **自訂技能**：使用者可在側邊欄新增自訂技能（輸入名稱、選擇分類含「其他」），支援放置、刪除，並持久化至 localStorage；匯出 / 匯入 JSON 亦包含自訂技能
- [ ] 多份矩陣管理（例如「現在的我」vs「半年後目標」）
- [ ] 分享功能：將結果編碼至網址，方便分享給他人查看
